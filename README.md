#  Monitor do Sistema em Python

Projeto desenvolvido para a disciplina de Sistemas Operacionais com o objetivo de monitorar recursos do computador em tempo real.

---

## 📌 Descrição

O programa coleta e exibe informações do sistema como uso da CPU, memória, processos e bateria.

Foi utilizado principalmente o módulo `psutil`, que permite acessar dados do sistema operacional diretamente pelo Python.

---

##  Objetivos

- Consultar informações do sistema
- Mostrar uso de CPU e memória
- Listar processos em execução
- Consultar processo por PID
- Tratar erros de execução

---

## 🛠️ Tecnologias

- Python
- psutil
- Tkinter
- Matplotlib

---

##  Funcionalidades

### ✔ Obrigatórias
- CPU (uso e núcleos)
- Memória (total, usada, disponível)
- Processos (PID e nome)
- Consulta por PID
- Bateria

###  Extras
- Gráfico de uso da CPU e memória
- Tempo de inicialização do sistema
- Informações de disco
- Rede (envio e recebimento)
- Usuários conectados
